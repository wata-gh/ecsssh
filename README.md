## Usage

```
$ ecsssh -r apne1 -c production-cluster -s your-awesome-product
```

Help

```
Usage: ecsssh [options]
    -h, --help                       Show help
    -r, --region VAL                 Specify AWS region for ECS
    -c, --cluster VAL                Specify cluster for ECS
    -s, --service VAL                Specify service for ECS
```
